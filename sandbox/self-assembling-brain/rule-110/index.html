Rule 110: Elementary Cellular Automaton

Rule 110 stands as one of the simplest known Turing complete systems—capable of universal computation through remarkably basic rules. Each cell looks at itself and its two neighbors, then decides its next state based on just 8 possible patterns. Yet from this minimal ruleset emerges complex behavior that can theoretically compute anything that's computable.

This cellular automaton perfectly illustrates how complex behaviors can emerge from simple rules—a key principle in understanding both neural development and artificial intelligence. Like neurons following local chemical signals to form elaborate networks, Rule 110's cells create intricate patterns through purely local interactions.

The Algorithm

At its core, Rule 110 operates on a beautifully simple principle of pattern matching:


// The 8 possible patterns and their outcomes
const RULE_110 = [
  // Pattern: [left, center, right] => new state
  0, // 000 -> 0
  1, // 001 -> 1
  1, // 010 -> 1
  1, // 011 -> 1
  0, // 100 -> 0
  1, // 101 -> 1
  1, // 110 -> 1
  0  // 111 -> 0
];
